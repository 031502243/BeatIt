# <span id="c"> 3. MarkDown文档编辑规范 </span>

## <span id="c.1"> 3.1 形成目录格式</span>
* 示例
    > * [1. 引言](#a)
    >   - [1.1 目的](#a.1)
    >   - [1.2 背景](#a.2)
    >   - [1.3 定义](#a.3)
    >   - [1.4 参考文献](#a.4)
    > * [2. 项目概述](#b)
    >   - [2.1 产品描述](#b.1)
    > # <span id = "a" > 1. 引言 </span>
    > ## <span id = "a.1" > 1.1 目的 </span>

* 代码
    ```
        > * [1. 引言](#a)
        >   - [1.1 目的](#a.1)
        >   - [1.2 背景](#a.2)
        >   - [1.3 定义](#a.3)
        >   - [1.4 参考文献](#a.4)
        > * [2. 项目概述](#b)
        >   - [2.1 产品描述](#b.1)
        > # <span id = "a" > 1. 引言 </span>
        > ## <span id = "a.1" > 1.1 目的 </span>       
    ```
    
## <span id="c.2">3.2 标题编辑</span>
* 示例

    # 一级标题( 1. 引言 )
    ## 二级标题 (1.1 目的)
    ### 三级标题 (1.1.1 第一点)

* 代码

    ```    
    # 一级标题( 1. 引言 )
    ## 二级标题 (1.1 目的)
    ### 三级标题 (1.1.1 第一点)
    ```

    > (3) .....

    
## <span id="c.4"> 3.4 段落格式 </span>
* 每个段落，首行行首缩进两个空格.

> $~~~~~~$这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例

    ```
    > $~~~~~~$这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例这是一个示例
    ```
